# Product

## Register

product

## Users
Developers, security analysts, and tech-savvy professionals who need to sanitize sensitive logs, databases, or configuration files (JSON, CSV, TXT) before sharing them with AI systems (e.g., ChatGPT, Gemini) or uploading them to public support tickets.

## Product Purpose
To provide a fast, secure, 100% local (client-side) tool to detect and mask Personally Identifiable Information (PII) and secret keys. The tool must ensure that no data is sent to external servers, providing maximum security with zero setup.

## Brand Personality
- **Clean**: Elegant, functional minimalism, focused on readability and flow.
- **Precise**: High information clarity, clear data state representation.
- **Trustworthy**: Understated professional aesthetic, clear local-only feedback.
- **3-Word personality**: Editorial, Refined, Secure.

## Anti-references
- ❌ **Cyberpunk neon / sci-fi HUDs**: Avoid glowing greens, grid overlays, or overly dramatic hacking aesthetics.
- ❌ **Generic SaaS cream/biscuit/sand defaults**: Do not use warm beige gradients or paper-textured backgrounds.
- ❌ **Side-stripe card borders**: No colored bars on card borders for category distinction.

## Design Principles
1. **Reduce Cognitive Load**: Present actions sequentially. The multi-step wizard guides the user through uploading, configuring, and verifying/exporting.
2. **Text Clarity Over Decoration**: Code and scanned inputs are the primary focal point. Spacing, typography, and contrast should highlight the data, not the UI borders.
3. **Local-First Reassurance**: Make it clear at every step that processing is 100% local, offline-capable, and private.
4. **State Coherency**: Masking rules should visually link to their targets so the user immediately understands how changing a rule affects the output.

## Accessibility & Inclusion
- Contrast ratio of at least 4.5:1 for body and code text.
- Keyboard navigability for the wizard progress and config chips.
- Respect `prefers-reduced-motion` for step transitions and loading states.
